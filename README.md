#   Miniguia SFN e Investimentos para Iniciantes

##  Sobre o Projeto
Este repositório contém um miniguia em Markdown sobre o **Sistema Financeiro Nacional (SFN)** e os principais tipos de investimentos.  
O objetivo é apoiar iniciantes na compreensão de conceitos básicos de finanças, com resumos, glossário e prompts de revisão.

## Como Usar
- Abra o arquivo **Miniguia-SFN-Investimentos.md** para acessar o conteúdo completo.  
- Navegue pelo **sumário** para ir direto às seções desejadas.  
- Consulte o **glossário** para revisar conceitos-chave.  
- Use os **prompts** para reforçar o aprendizado com IA ou revisões rápidas.
